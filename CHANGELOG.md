# Advanced Java Folding IntelliJ Plugin Changelog

## [Unreleased]

## [1.0.6]

### Added

- Fold Optional.ofNullable, Optional.of, Optional.orElseThrow & Optional.flatMap

### Added

- Display Optional as Kotlin Null-Safe
